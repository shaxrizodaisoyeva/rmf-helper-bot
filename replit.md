# Reflektiv Med Farm Bot

Telegram bot for pharmaceutical product distribution management.

## Project Structure
- `main.py`: Main entry point for the Telegram bot.
- `keep_alive.py`: Flask server to keep the bot alive.
- `contacts_image.jpg`: Image for the contacts section.
- `*.xlsx`: Price lists for various regions and teams.
- `*.pdf`: Product leaflets and certificates.

## Setup
1. Set `TELEGRAM_TOKEN` environment variable.
2. Run `python main.py`.

## Workflows
- `run`: Starts the Telegram bot and the keep-alive server.
